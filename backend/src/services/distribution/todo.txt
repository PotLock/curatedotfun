# Simplified Distribution Service Architecture

## Core Concepts

1. JSON-Driven Configuration
```typescript
interface PluginConfig {
  // Core plugin metadata
  name: string;
  version: string;
  type: 'transform' | 'distribute';
  
  // Plugin capabilities
  routes?: {
    path: string;
    method: 'GET' | 'POST';
    description: string;
  }[];
  
  // Data handling
  schema: {
    input: JSONSchema;  // JSON Schema for input validation
    output?: JSONSchema; // JSON Schema for output validation (transforms)
    config: JSONSchema;  // JSON Schema for plugin configuration
  };
  
  // AI assistance
  examples: {
    input: unknown;
    output?: unknown;
    config: Record<string, unknown>;
    description: string;
  }[];
}
```

2. Simplified Plugin Interface
```typescript
interface Plugin {
  // Core methods
  initialize(config: Record<string, unknown>): Promise<void>;
  
  // Optional route handler
  handleRoute?(ctx: Hono.Context): Promise<Response>;
  
  // Plugin-specific actions
  transform?(input: unknown): Promise<unknown>;
  distribute?(input: unknown): Promise<void>;
}
```

## Recommended Packages

1. Core Framework:
- hono - Web framework
  * Pros: Fast, lightweight, great TypeScript support
  * Use case: API routes, middleware

2. Queue System:
- graphile-worker - PostgreSQL-based job queue
  * Pros: Uses existing Postgres, reliable, typed
  * Use case: Background jobs, retries

3. Validation:
- zod - Schema validation
  * Pros: TypeScript integration, JSON Schema support
  * Use case: Config validation, API inputs

4. Plugin System:
- @module-federation/node - Dynamic loading
  * Pros: Hot reloading, versioning
  * Use case: Plugin loading

## Example Plugin Configuration

```json
{
  "name": "telegram-distributor",
  "version": "1.0.0",
  "type": "distribute",
  
  "routes": [
    {
      "path": "/webhook",
      "method": "POST",
      "description": "Telegram webhook endpoint"
    }
  ],
  
  "schema": {
    "input": {
      "type": "object",
      "properties": {
        "content": { "type": "string" },
        "media": { 
          "type": "array",
          "items": { "type": "string", "format": "uri" }
        }
      }
    },
    "config": {
      "type": "object",
      "required": ["botToken", "channelId"],
      "properties": {
        "botToken": { "type": "string" },
        "channelId": { "type": "string" }
      }
    }
  },
  
  "examples": [
    {
      "description": "Send text message",
      "input": {
        "content": "Hello world!"
      },
      "config": {
        "botToken": "123:ABC",
        "channelId": "@channel"
      }
    }
  ]
}
```
